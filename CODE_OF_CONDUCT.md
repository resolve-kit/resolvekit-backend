# Code of Conduct

## Our Standard

We want this project to be a practical, respectful place to collaborate. Participants are expected to communicate clearly, assume good intent, and focus criticism on ideas, code, and outcomes.

Unacceptable behavior includes harassment, threats, personal attacks, discriminatory language, deliberate disruption, or publishing private information without consent.

## Enforcement

Maintainers may remove comments, close issues or pull requests, restrict participation, or ban contributors when behavior harms the project or its participants.

If you need to report a conduct issue, contact the maintainers through the private channel listed on the repository once one is configured. If the issue involves a security vulnerability, follow [SECURITY.md](SECURITY.md) instead.
