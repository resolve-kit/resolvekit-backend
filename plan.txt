 iOS App Agent - Backend Service Design & Implementation Plan                                                                                                                                                      │
│                                                                                                                                                                                                                   │
│ Context                                                                                                                                                                                                           │
│                                                                                                                                                                                                                   │
│ Build a backend service for a product where iOS app developers integrate a Swift SDK into their apps. The SDK registers locally-callable functions and communicates with a backend-hosted LLM agent. The backend  │
│ orchestrates which functions to call on the device, in what order, based on natural language chat from end users.                                                                                                 │
│                                                                                                                                                                                                                   │
│ Core flow: iOS SDK registers functions → developer configures agent per app on backend → end user chats → LLM decides function calls → iOS app executes locally → results fed back to LLM → final response.       │
│                                                                                                                                                                                                                   │
│ Key decisions from user:                                                                                                                                                                                          │
│ - Multi-app: One developer account can have multiple apps, each with its own config, functions, and sessions                                                                                                      │
│ - Web dashboard: Include a developer-facing web UI for configuration                                                                                                                                              │
│ - SDK interface: Define the Swift SDK's public API surface (protocols, types, delegate callbacks)                                                                                                                 │
│                                                                                                                                                                                                                   │
│ ---                                                                                                                                                                                                               │
│ Architecture Overview                                                                                                                                                                                             │
│                                                                                                                                                                                                                   │
│ - Framework: FastAPI (async, WebSocket support, auto OpenAPI docs)                                                                                                                                                │
│ - Database: PostgreSQL via asyncpg + SQLAlchemy 2.0 async                                                                                                                                                         │
│ - LLM Layer: LiteLLM (unified interface to OpenAI, Anthropic, Google, etc.)                                                                                                                                       │
│ - Real-time: WebSocket primary (bidirectional needed for tool call/result flow), HTTP SSE fallback                                                                                                                │
│ - Auth: API keys per app (prefixed iaa_, SHA-256 hashed); JWT for developer dashboard                                                                                                                             │
│ - Encryption: Fernet for LLM API keys at rest                                                                                                                                                                     │
│ - Dashboard: React + Vite SPA served separately, consumes the same REST API via JWT auth                                                                                                                          │

│ ---                                                                                                                                                                                                               │
│ WebSocket Protocol                                                                                                                                                                                                │
│                                                                                                                                                                                                                   │
│ All messages use a typed envelope: {type, request_id, payload, timestamp}                                                                                                                                         │
│                                                                                                                                                                                                                   │
│ Client → Server                                                                                                                                                                                                   │
│                                                                                                                                                                                                                   │
│ - chat_message — {text: string}                                                                                                                                                                                   │
│ - tool_result — {call_id, status: "success"|"error", result|error}                                                                                                                                                │
│ - ping                                                                                                                                                                                                            │
│                                                                                                                                                                                                                   │
│ Server → Client                                                                                                                                                                                                   │
│                                                                                                                                                                                                                   │
│ - assistant_text_delta — streamed text chunks {delta, accumulated}                                                                                                                                                │
│ - tool_call_request — {call_id, function_name, arguments, timeout_seconds}                                                                                                                                        │
│ - turn_complete — {full_text, usage}                                                                                                                                                                              │
│ - error — {code, message, recoverable}                                                                                                                                                                            │
│ - pong                                                                                                                                                                                                            │
│                                                                                                                                                                                                                   │
│ Multi-turn flow                                                                                                                                                                                                   │
│                                                                                                                                                                                                                   │
│ User sends chat_message                                                                                                                                                                                           │
│   → Backend calls LLM with context + tools                                                                                                                                                                        │
│   → LLM returns tool_calls                                                                                                                                                                                        │
│     → Backend sends tool_call_request(s) to iOS                                                                                                                                                                   │
│     → iOS executes, sends tool_result(s) back                                                                                                                                                                     │
│     → Backend feeds results to LLM, loops                                                                                                                                                                         │
│   → LLM returns text                                                                                                                                                                                              │
│     → Backend streams assistant_text_delta                                                                                                                                                                        │
│     → Backend sends turn_complete                                                                                                                                                                                 │
│                                                                                                                                                                                                                   │
│ Parallel tool calls: multiple tool_call_request messages sent, server waits for all results before next LLM call.                                                                                                 │
│                                                                                                                                                                                                                   │
│ ---                                                                                                                                                                                                               │
│ Core Orchestrator Logic                                                                                                                                                                                           │
│                                                                                                                                                                                                                   │
│ The agent loop in services/orchestrator.py:                                                                                                                                                                       │
│ 1. Persist user message, load context window (last N messages + system prompt)                                                                                                                                    │
│ 2. Call LLM with messages + tools (streaming)                                                                                                                                                                     │
│ 3. If LLM returns text → stream to client, save, send turn_complete                                                                                                                                               │
│ 4. If LLM returns tool_calls → send tool_call_request to iOS, wait for results with per-function timeouts                                                                                                         │
│ 5. Append results to context, repeat from step 2                                                                                                                                                                  │
│ 6. Safety: max_tool_rounds cap (default 10), then force text response                                                                                                                                             │
│                                                                                                                                                                                                                   │

│ // The main entry point — initialized once in the app                                                                                                                                                             │
│ public class AppAgent {                                                                                                                                                                                           │
│     public init(apiKey: String, baseURL: URL = .defaultBackendURL)                                                                                                                                                │
│                                                                                                                                                                                                                   │
│     /// Register all callable functions. Call on app launch.                                                                                                                                                      │
│     /// Bulk-syncs with backend (deactivates any previously registered                                                                                                                                            │
│     /// functions not in this list).                                                                                                                                                                              │
│     public func registerFunctions(_ functions: [AgentFunction]) async throws                                                                                                                                      │
│                                                                                                                                                                                                                   │
│     /// Create a new chat session for the current user/device.                                                                                                                                                    │
│     public func createSession(                                                                                                                                                                                    │
│         deviceId: String? = nil,    // defaults to UIDevice identifierForVendor                                                                                                                                   │
│         metadata: [String: String] = [:]                                                                                                                                                                          │
│     ) async throws -> ChatSession                                                                                                                                                                                 │
│ }                                                                                                                                                                                                                 │
│                                                                                                                                                                                                                   │
│ // Defines a function the LLM can call on-device                                                                                                                                                                  │
│ public struct AgentFunction {                                                                                                                                                                                     │
│     public let name: String                                                                                                                                                                                       │
│     public let description: String                                                                                                                                                                                │
│     public let parameters: [FunctionParameter]                                                                                                                                                                    │
│     public let timeoutSeconds: Int                                                                                                                                                                                │
│     public let handler: ([String: Any]) async throws -> Any                                                                                                                                                       │
│                                                                                                                                                                                                                   │
│     public init(                                                                                                                                                                                                  │
│         name: String,                                                                                                                                                                                             │
│         description: String,                                                                                                                                                                                      │
│         parameters: [FunctionParameter] = [],                                                                                                                                                                     │
│         timeoutSeconds: Int = 30,                                                                                                                                                                                 │
│         handler: @escaping ([String: Any]) async throws -> Any                                                                                                                                                    │
│     )                                                                                                                                                                                                             │
│ }                                                                                                                                                                                                                 │
│                                                                                                                                                                                                                   │
│ public struct FunctionParameter {                                                                                                                                                                                 │
│     public let name: String                                                                                                                                                                                       │
│     public let type: ParameterType       // .string, .int, .double, .bool, .array, .object                                                                                                                        │
│     public let description: String                                                                                                                                                                                │
│     public let required: Bool                                                                                                                                                                                     │
│     public let enumValues: [String]?     // for restricted string values                                                                                                                                          │
│ }                                                                                                                                                                                                                 │

│ Chat Session                                                                                                                                                                                                      │
│                                                                                                                                                                                                                   │
│ public class ChatSession {                                                                                                                                                                                        │
│     public let id: UUID                                                                                                                                                                                           │
│     public var delegate: ChatSessionDelegate?                                                                                                                                                                     │
│                                                                                                                                                                                                                   │
│     /// Send a user message. Responses arrive via delegate callbacks.                                                                                                                                             │
│     public func send(message: String) async throws                                                                                                                                                                │
│                                                                                                                                                                                                                   │
│     /// Close the session.                                                                                                                                                                                        │
│     public func close() async throws                                                                                                                                                                              │
│ }                                                                                                                                                                                                                 │
│                                                                                                                                                                                                                   │
│ public protocol ChatSessionDelegate: AnyObject {                                                                                                                                                                  │
│     /// Streaming text chunk from the assistant                                                                                                                                                                   │
│     func session(_ session: ChatSession, didReceiveTextDelta delta: String, accumulated: String)                                                                                                                  │
│                                                                                                                                                                                                                   │
│     /// Assistant's turn is complete                                                                                                                                                                              │
│     func session(_ session: ChatSession, didCompleteTurn fullText: String)                                                                                                                                        │
│                                                                                                                                                                                                                   │
│     /// An error occurred                                                                                                                                                                                         │
│     func session(_ session: ChatSession, didEncounterError error: AgentError)                                                                                                                                     │
│ }                                                                                                                                                                                                                 │
│                                                                                                                                                                                                                   │
│ How it works internally                                                                                                                                                                                           │
│                                                                                                                                                                                                                   │
│ - registerFunctions() converts AgentFunction array to JSON Schema format, calls PUT /v1/functions/bulk                                                                                                            │
│ - createSession() calls POST /v1/sessions, opens WebSocket to returned ws_url                                                                                                                                     │
│ - send(message:) sends chat_message over WebSocket                                                                                                                                                                │
│ - When tool_call_request arrives, SDK finds the matching AgentFunction.handler, executes it locally, sends tool_result back — all transparent to the app developer                                                │
│ - Text deltas and turn completions are forwarded to ChatSessionDelegate                                                                                                                                           │

│ let agent = AppAgent(apiKey: "iaa_xxxxx")                                                                                                                                                                         │
│                                                                                                                                                                                                                   │
│ // Register functions on launch                                                                                                                                                                                   │
│ try await agent.registerFunctions([                                                                                                                                                                               │
│     AgentFunction(                                                                                                                                                                                                │
│         name: "set_alarm",                                                                                                                                                                                        │
│         description: "Set an alarm on the user's device",                                                                                                                                                         │
│         parameters: [                                                                                                                                                                                             │
│             .init(name: "time", type: .string, description: "ISO 8601 time", required: true),                                                                                                                     │
│             .init(name: "label", type: .string, description: "Alarm label", required: false),                                                                                                                     │
│         ],                                                                                                                                                                                                        │
│         handler: { args in                                                                                                                                                                                        │
│             let time = args["time"] as! String                                                                                                                                                                    │
│             let label = args["label"] as? String                                                                                                                                                                  │
│             let alarm = try await AlarmManager.shared.setAlarm(time: time, label: label)                                                                                                                          │
│             return ["alarm_id": alarm.id, "scheduled_time": alarm.scheduledTime]                                                                                                                                  │
│         }                                                                                                                                                                                                         │
│     ),                                                                                                                                                                                                            │
│     AgentFunction(                                                                                                                                                                                                │
│         name: "get_health_data",                                                                                                                                                                                  │
│         description: "Read health metrics from HealthKit",                                                                                                                                                        │
│         parameters: [                                                                                                                                                                                             │
│             .init(name: "metric", type: .string, description: "steps, heart_rate, or sleep", required: true, enumValues: ["steps", "heart_rate", "sleep"]),                                                       │
│         ],                                                                                                                                                                                                        │
│         handler: { args in                                                                                                                                                                                        │
│             let metric = args["metric"] as! String                                                                                                                                                                │
│             return try await HealthManager.shared.getData(metric: metric)                                                                                                                                         │
│         }                                                                                                                                                                                                         │
│     )                                                                                                                                                                                                             │
│ ])                                                                                                                                                                                                                │
│                                                                                                                                                                                                                   │
│ // Start a chat session                                                                                                                                                                                           │
│ let session = try await agent.createSession()                                                                                                                                                                     │
│ session.delegate = self                                                                                                                                                                                           │
│                                                                                                                                                                                                                   │
│ // User taps send                                                                                                                                                                                                 │
│ try await session.send(message: "Set an alarm for 7am and show me my steps today")                                                                                                                                │
│ // SDK handles everything: LLM decides to call set_alarm + get_health_data,                                                                                                                                       │
│ // SDK executes handlers locally, sends results back, LLM responds with summary                                                                                                                                   │

│ Error Handling                                                                                                                                                                                                    │
│                                                                                                                                                                                                                   │
│ ┌───────────────────────────┬────────────────────────────────────────────────────────┐                                                                                                                            │
│ │         Scenario          │                        Handling                        │                                                                                                                            │
│ ├───────────────────────────┼────────────────────────────────────────────────────────┤                                                                                                                            │
│ │ Function timeout          │ Synthesize error result, feed to LLM                   │                                                                                                                            │
│ ├───────────────────────────┼────────────────────────────────────────────────────────┤                                                                                                                            │
│ │ Function error            │ SDK sends status: "error", passed to LLM as context    │                                                                                                                            │
│ ├───────────────────────────┼────────────────────────────────────────────────────────┤                                                                                                                            │
│ │ LLM API error             │ Send error message, client can retry                   │                                                                                                                            │
│ ├───────────────────────────┼────────────────────────────────────────────────────────┤                                                                                                                            │
│ │ Unknown function from LLM │ Error result: "Unknown function, available: [...]"     │                                                                                                                            │
│ ├───────────────────────────┼────────────────────────────────────────────────────────┤                                                                                                                            │
│ │ Concurrent messages       │ Per-session async lock, reject with turn_in_progress   │                                                                                                                            │
│ ├───────────────────────────┼────────────────────────────────────────────────────────┤                                                                                                                            │
│ │ WebSocket disconnect      │ Save partial state, session stays active for reconnect │                                                                                                                            │
│ ├───────────────────────────┼────────────────────────────────────────────────────────┤                                                                                                                            │
│ │ Context too large         │ Sliding window truncation on message count             │                                                                                                                            │
│ └───────────────────────────┴────────────────────────────────────────────────────────┘                                                                                                                            │
